# Toggl iCal

## Useage

1. Open Calendar
2. `File` -> `New Calendar Subscription...`
3. Paste `https://toggl-ical.kattcorp.com/?token=your_toggl_token_here`

4. Open Calendar
5. `File` -> `New Calendar Subscription...`
6. Paste `http://localhost:3000/?token=your_toggl_token_here`
